Compile time flags in Rust

Rust like C or C++ allows programmers to set flags during compile time. These flags can be grouped into features that can be set via the cargo command.

Sometimes you might want to set only a single flag and not enable a feature.

To do so, you need to set the flag via .cargo/config in your project’s source code.

Here is an example of a simple flag to conditionally compile code. 

#[cfg(debug)]
fn debug(s: &str) {
    println!("string is stan {}", s);
}

#[cfg(not(debug))]
fn debug(_s: &str) {
}

To set the debug flag during compilation, modify the config file for cargo 

[target.x86_64-unknown-linux-gnu]
rustflags = ["--cfg", "debug"]

The reason we have to set it via the config file is that, Cargo doesn’t allow setting arbitrary flags via the cargo command.
